## Списки

### Нумерованный список:
1. Первый пункт
2. Второй пункт
   3. Вложенный пункт
4. Третий пункт

### Маркированный список:
- Пункт 1
- Пункт 2
  - Вложенный маркированный список
- Пункт 3

### Список задач (GitHub-style):
- [x] Выполненная задача
- [ ] Не выполненная задача

## Код и примеры

Встроенный код: `print("Hello, world!")`

Блок кода на Python:
```python
def hello():
    print("Hello, world!")
```

Блок кода на JavaScript:
```javascript
function hello() {
    console.log("Hello, world!");
}
```
